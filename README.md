## Sortmancer

Personal program that scans old stock photo/texture CDs and uses image recognition models to label them properly (which is useful for ones that have pretty poor labelling).

<img width="1408" height="1001" alt="image" src="https://github.com/user-attachments/assets/0f43bf12-c7a5-4ce7-a866-f4ce5de4f462" />

Currently only supports on Linux, there is a hole left for Windows support and if anyone else wants to actually use this program I might do it (or PRs are open if you want to).

**Note that on Linux, this hard depends on:**
- sqlite3
- openssl/libcrypto
- libblkid

Any other dependencies this uses are statically linked.
